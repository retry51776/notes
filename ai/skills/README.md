# Skills

## Skill Design

<https://lavinigam.com/posts/adk-skill-design-patterns/>

5 main skill categories:

- Tool Wrapper
- Generator
- Reviewer
- Inversion ~ `Input Form before workflow`
- Pipeline ~ `chain workflows`

## Infographic Designer

- [infographic-designer](./infographic-designer/SKILL.md): A reusable
  infographic skill for turning source material or a topic into a
  single-image visual explainer with a clear narrative, visual
  hierarchy, and production-ready prompt or spec.

## Personal Eval

- [personal-eval](./personal-eval/SKILL.md): A structured self-evaluation
  skill that collects user context first, then applies inversion-based
  layered analysis.

## Pitfalls

- [pitfalls](./pitfalls/SKILL.md): A reusable skill for common agent
  mistakes, recovery moves, and execution tips.

## PR Reviewer

- [pr-reviewer](./pr-reviewer/SKILL.md): A pre-push review skill for
  finding bugs, regressions, missing tests, and risky changes before a
  PR is pushed.

## Social Simulator

- [social-simulator](./social-simulator/SKILL.md): A structured
  conversation simulator for communication and social cognition
  training.
