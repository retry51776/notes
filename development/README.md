- Cache
  - Capacity
  - Latency
  - Throughput

    - increase cache hit rate
      - change execute order

- Computation

  - Interrupt `swap tasks to increase hardware efficiency`
  - Predict `trade/waste computation for speed`
  - Arithmetic
